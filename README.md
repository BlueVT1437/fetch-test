## Installation

FETCH LUMENS requires [Node.js](https://nodejs.org/) v17+ to run.

Install the dependencies and devDependencies and start the server.

```sh
cd fetch_lumens
git checkout develop
npm i
npm run start
```

```sh
http://localhost:3000/home
```
